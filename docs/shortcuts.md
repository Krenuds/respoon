# Keyboard Shortcuts

Complete reference for all keyboard shortcuts and mouse controls in Respoon.

## Opening the Editor

| Input | Action                     |
| ----- | -------------------------- |
| `/rs` | Chat command to toggle the editor open or closed |

The editor can also be bound to a key in **FiveM Settings > Key Bindings > Respoon**. The keybind is unbound by default.

## Transform Controls

These shortcuts control how you manipulate entities in the 3D viewport.

| Key   | Action                                    |
| ----- | ----------------------------------------- |
| W     | Translate mode (move gizmo)               |
| E     | Rotate mode (rotate gizmo)                |
| Q     | Toggle between World and Local coordinate space |
| Space | Snap selected entity to ground            |

**Translate mode** moves the entity along the X, Y, or Z axis. **Rotate mode** rotates it. Press **Q** to switch between world-aligned axes and the entity's local axes.

## Entity Operations

| Key      | Action                                      |
| -------- | ------------------------------------------- |
| Ctrl+D   | Clone the selected entity or particle       |
| Delete   | Delete the selected entity or particle      |
| Enter    | Confirm entity preview (spawn the entity)   |
| Escape   | Cancel preview or close the editor          |

When browsing the model catalog, selecting a model places it in preview mode. Press **Enter** to confirm and spawn it, or **Escape** to cancel.

## Scene Playback

| Key       | Action                                  |
| --------- | --------------------------------------- |
| PageDown  | Play all task sequences from the start  |
| End       | Stop all running tasks                  |
| Delete    | Reset entity positions (HUD mode only)  |

Scene playback controls execute task sequences assigned to entities. Use **PageDown** to start playback and **End** to stop it.

## Recording

| Key       | Action                |
| --------- | --------------------- |
| PageUp    | Start clip recording  |
| Insert    | Stop clip recording   |

Recording captures the current scene for playback or export.

## UI Controls

| Key  | Action                                            |
| ---- | ------------------------------------------------- |
| Home | Toggle HUD strip (minimal playback-only interface) |

The HUD strip is a compact overlay that provides scene playback controls without the full editor UI.

## Mouse Controls

| Input       | Action                                 |
| ----------- | -------------------------------------- |
| Left Click  | Select an entity in the viewport       |
| Right Click | Open the context menu                  |
| Drag Gizmo  | Move or rotate the selected entity     |
| Hover       | Preview entity information             |

Left-click an entity to select it and display its gizmo handles. Drag the gizmo arrows or rings to transform the entity. Right-click for additional options like cloning, deleting, or editing properties.

## Rebinding Keys

All keybindings can be customized through FiveM:

1. Open the FiveM pause menu
2. Go to **Settings > Key Bindings**
3. Scroll to the **Respoon** section
4. Click on any binding to reassign it

Custom bindings persist across sessions and are stored in your FiveM client settings.
